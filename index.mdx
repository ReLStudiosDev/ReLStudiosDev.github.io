# WhackerLinkFiveM: Getting Started

WhackerLinkFiveM is a FiveM resource that provides realistic P25 digital radio communications for roleplay servers. This client-side interface connects to WhackerLink servers to enable authentic radio experiences within FiveM.

## üöÄ Key Features

- **Realistic P25 Radio Models** - Support for multiple radio models including APX series, SRX scanners, and mobile units
- **Voice Communications** - Full duplex voice transmission with realistic radio effects
- **Emergency Functions** - Emergency alarms, site trunking, and emergency communications
- **Scanning Capabilities** - Multi-channel scanning with priority handling
- **Location Services** - GPS integration for location-based features
- **Configurable Codeplugs** - Customizable radio configurations and talk groups

## üìã Prerequisites

Before installing WhackerLinkFiveM, ensure you have:

- A running WhackerLink v4 server instance
- FiveM server with appropriate permissions
- Basic understanding of FiveM resource management

## üí° Quick Start

1. **Download** the latest WhackerLinkFiveM release from [GitHub](https://github.com/WhackerLink/WhackerLinkFiveM)
2. **Extract** to your FiveM server's `resources` folder
3. **Configure** your codeplugs and server connections
4. **Start** the resource and connect players to your WhackerLink server
5. **Test** radio communications in-game
